# global.livin.in

One Platform. Every Role. One Feed.

The unified LIVIN Media platform — recruiting, city management, community, and AI in one place.

## Tech Stack
- Next.js 15 + React 19 + Tailwind v4
- Supabase Auth (hub) + 3 Supabase projects
- Supabase Realtime for live feed + chat

## Development
```bash
npm install
cp .env.example .env.local
npm run dev
```
